# Tools Introduction

This directory contains Jupyter notebooks designed to introduce the core technologies used in this project.

## Contents

- `huggingface_intro.ipynb`: Introduction to the Hugging Face `transformers` and `datasets` libraries.
- `networkx_intro.ipynb`: Tutorial on using `NetworkX` for graph manipulation and visualization.
- `openai_intro.ipynb`: Guide to using the OpenAI API (and compatible APIs like OpenRouter) for LLM interactions.
